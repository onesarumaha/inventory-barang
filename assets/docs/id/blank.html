Halaman Kosong

  • Home
  • Halaman Kosong

Membuat halaman Kosong, Pertama kamu bisa copy paste Struktur Dasar Template yang ada dihalaman Instalasi, lalu cocokkan url yang ada dengan struktur aplikasi atau web kamu. Membuat halaman Kosong tidaklah sulit, jika sebelumnya kamu sudah membaca dokumentasi Navbar, Sidebar, Main dan Footer, sebenarnya halaman Kosong adalah halaman yang hanya ada Navbar, Sidebar, Main dan Footer nya saja, serta didalam Main hanya terdapat Main Header.
