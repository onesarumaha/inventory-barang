Info Box

  • Home
  • Info Box

Membuat Info Box kamu membutuhkan tag <section> atau <div> dengan class .info-box dan class untuk style Info Box-nya, ada class .info-box--blue, .info-box--green, .info-box--orange dan .info-box--red. Didalamnya ada beberapa bagian, yaitu Icon, Deskripsi, dan Button.

Pertama, Icon. Membuat Icon kamu membutuhkan tag <div> dengan class .info-box__icon, didalamnya kamu bisa meletakkan icon dari Font Awesome menggunakan tag <span>.

Kedua, Deskripsi. Membuat Deskripsi kamu membutuhkan tag <div> dengan class .info-box__description. Didalamnya kamu bisa menambahkan judul dari Info Box dengan tag <h2>. Kemudian dibawah judul kamu bisa menambahkan semacam informasi utama dari judul menggunakan tag <h1>, misalnya judul Info Box adalah Anggota maka informasi utamanya adalah jumlah Anggota, yang biasanya ditulis dalam bentuk angka. Terakhir dibawah informasi utama kamu juga bisa menambahkan waktu dengan menggunakan tag <time>. Dan yang paling penting letakkan Deskripsi setelah Icon!.

Ketiga, Button. Membuat button untuk detail dari Info Box, kamu hanya perlu tag <a> dengan class .info-box__btn-detail, didalamnya kamu bisa meletakkan icon dari font awesome menggunakan tag <span>. Dan jangan lupa untuk meletakkan Button setelah Deskripsi!.

Jika kamu merasa Info Box-nya terlalu lebar, kamu bisa meletakkannya didalam grid gistem Bootstrap.
