Halaman Daftar

  • Home
  • Halaman Daftar

Membuat halaman Daftar, Pertama kamu bisa copy paste Struktur Dasar Template yang ada di bagian Instalasi, lalu cocokkan url yang ada dengan struktur aplikasi atau web kamu, dan untuk membuat halaman Daftar, standarnya kamu tidak perlu meload javascript apapun.

Kedua, didalam tag <body> buat element menggunakan tag <div> dengan class .container-fluid, lalu didalam element <div class="container-fluid"> buat element menggunakan tag <div> dengan class .register, didalamnya ada dua bagian yaitu Daftar Konten dan Button Login. Untuk membuat Daftar Konten kamu perlu tag <div> dengan class .register__content. Untuk membuat Button Login kamu perlu tag <a> dengan class .btn diikuti dengan class untuk style Buttonnya .btn--link. Letakkan Button Login setelah Daftar Konten.

Ketiga, didalam Daftar Konten ada beberapa bagian yaitu Daftar Head (tempat logo dan keterangan daftar), Daftar Form, dan Daftar Alt (tempat button daftar dengan Facebook atau Google).

Untuk membuat Daftar Head, kamu perlu tag <div> dengan class .register__head, didalamnya kamu bisa menggunakan tag <img> untuk menampilkan logo, dan gunakan tag <h5> atau <p> untuk menambahkan keterangan daftar.

Untuk membuat Daftar From, kamu perlu tag <div> dengan class .register__form lalu didalamnya buat element menggunakan tag <form>, kemudian didalam tag <form> kamu bisa membuat Form seperti pada dokumentasi Form dan menambahkan Button seperti pada dokumentasi Button, namun kamu harus membungkus Button-nya menggunakan tag <div> dengan class .register__form-action. Letakkan Daftar Form setelah Daftar Head.

Untuk membuat Daftar Alt, kamu perlu tag <div> dengan class .register__alt, lalu didalamnya kamu bisa membuat 'Button Daftar dengan Facebook' menggunakan tag <a> dengan class .btn diikuti dengan class untuk style Buttonnya .btn--register-alt-facebook. Untuk membuat 'Button Daftar dengan Google' caranya hampir sama dengan membuat 'Button Daftar dengan Facebook', hanya saja kamu perlu mengubah class untuk style Buttonnya menjadi .btn--register-alt-google. Letakkan Login Alt setelah Daftar Form.
