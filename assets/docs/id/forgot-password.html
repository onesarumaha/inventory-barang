Halaman Lupa Password

  • Home
  • Halaman Lupa Password

Membuat halaman Lupa Password, Pertama kamu bisa copy paste Struktur Dasar Template yang ada di bagian Instalasi, lalu cocokkan url yang ada dengan struktur aplikasi atau web kamu, dan untuk membuat halaman Login, standarnya kamu tidak perlu meload javascript apapun.

Kedua, didalam tag <body> buat element menggunakan tag <div> dengan class .container-fluid. Lalu didalam element <div class="container-fluid"> buat element menggunakan tag <div> dengan class .forgot-password, didalamnya ada tiga bagian yaitu, Lupa Password Konten, Button Buat Akun, dan Button Login. Untuk membuat Lupa Password Konten kamu perlu tag <div> dengan class .forgot-password__content. Untuk membuat Button Buat Akun dan Button Login, kamu perlu tag <a> dengan class .btn diikuti dengan class untuk style Buttonnya .btn--link, letakkan Button Buat Akun dan Button Login setelah Lupa Password Konten.

Ketiga, didalam Lupa Password Konten ada dua bagian yaitu Lupa Password Head (tempat logo dan keterangan Lupa Password) dan Lupa Password Form.

Untuk membuat Lupa Password Head kamu perlu tag <div> dengan class .forgot-password__head, lalu didalamnya kamu bisa memasukkan logo dengan menggunakan tag <img>, dan gunakan tag <p> untuk menambahkan keterangan Lupa Password.

Untuk membuat Lupa Password Form kamu perlu tag <div> dengan class .forgot-password__form didalamnya buat element menggunakan tag <form>, kemudian didalam tag <form> kamu bisa menambahkan Form seperti pada dokumentasi Form dan Button seperti pada dokumentasi Button. Letakkan Lupa Password Form setelah Lupa Password Head.
